# GameRadar 품질 개선 로드맵

> 게임 뉴스 데이터 수집 품질 향상을 위한 실행 계획

## 현황 요약

| 지표 | 현재 값 | 상태 |
|------|--------|------|
| 소스 수 | 45 | 양호 |
| 매칭률 | 78% | 양호 |
| 테스트 수 | 23개 | 양호 |

---

## P0: 긴급 (없음)

현재 긴급 수정이 필요한 사항 없음. 시스템 건강 상태 양호.

---

## P1: 단기 개선 (1-2주)

### 한국 게임 뉴스 소스 추가

**목표**: 국내 게임 시장 커버리지 강화

**작업 내용**:
- [ ] 국내 주요 게임 미디어 추가
  - 인벤
  - 루리웹
  - 게임메카
  - 게임동아
  - 게임조선
- [ ] 게임사 공식 채널 추가
  - 넥슨 뉴스룸
  - 엔씨소프트 뉴스
  - 크래프톤 뉴스
  - 넷마블 뉴스
  - 카카오게임즈 뉴스
- [ ] 게임 커뮤니티 RSS/피드 추가
  - 디시인사이드 게임 갤러리 인기글
  - 에펨코리아 게임 게시판
- [ ] 게임 유튜버 채널 연동 검토

**검증 기준**:
- 신규 한국 소스 15개 이상 추가
- 한국어 콘텐츠 비율 50% 이상 달성
- 소스 성공률 95% 이상 유지

---

## P2: 중기 개선 (3-4주)

### 스팀 스토어 연동

**목표**: 게임 출시/할인 정보 실시간 수집

**작업 내용**:
- [ ] Steam Web API 연동
- [ ] 신규 출시 게임 목록 자동 수집
- [ ] 할인 정보 및 가격 변동 추적
- [ ] Steam 인기 게임 순위 수집
- [ ] Steam 리뷰 요약 데이터 수집

### 플레이스테이션 스토어 연동

**목표**: 콘솔 게임 정보 커버리지 확대

**작업 내용**:
- [ ] PlayStation Store API 연동 조사
- [ ] PS 독점작 출시 정보 수집
- [ ] PS Plus 월간 무료 게임 정보 수집
- [ ] PS 할인 행사 정보 수집

### 추가 플랫폼 연동 (선택)

**목표**: 멀티플랫폼 커버리지 완성

**작업 내용**:
- [ ] Xbox Game Pass 정보 수집
- [ ] Nintendo eShop 할인 정보 수집
- [ ] Epic Games Store 무료 게임 정보 수집

**검증 기준**:
- 스토어 데이터 일일 자동 수집
- 신규 출시작 정보 24시간 내 반영
- 가격 변동 알림 시스템 구축
